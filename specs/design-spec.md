# Storymate Landing Page — Design Spec

## Goal
Create a landing page for "Storymate", an AI-powered children's storybook creator. The page must be visually engaging, cartoony, and friendly for children and parents.

## Target Audience
- Children (ages 4-10)
- Parents looking for creative tools for their kids

## Visual Style
- **Theme:** Storybook / Magical / Cartoony
- **Color Palette:** Bright, soft colors (pastels, sky blue, playful yellows)
- **Imagery:** 3 core hero images (e.g., a child reading with a friendly robot, a magical forest scene, a "Story Creator" interface mockup)

## Page Sections
1. **Hero Header:** Catchy title ("Bring Your Imagination to Life"), CTA ("Start Creating"), and 1 Hero Image.
2. **How It Works:** 3-step visual guide (1. Imagine, 2. AI Writes, 3. You Read).
3. **Features:** Cartoon icons for "Personalized Stories," "Cute Illustrations," "Read Together."
4. **Gallery:** Mockups of generated storybooks.
5. **Footer:** Simple links and contact.

## Tech Stack
- React + Vite
- Tailwind CSS
- Lucide Icons
- Framer Motion (optional animations)

## Deliverables
- Source code in `/shared-workspaces/projects/storymate/src/`
- 3 images in `/shared-workspaces/projects/storymate/assets/`
- Coolify deployment configuration in `/shared-workspaces/projects/storymate/infra/`
